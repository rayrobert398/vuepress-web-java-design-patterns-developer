# Java Design Patterns web site

This repository contains the source code of Java Design Patterns web site, powered by [Vuepress](https://vuepress.vuejs.org/).

Running it locally is easy, just follow these steps

- Clone this repository
- Change directory `cd docs`
- Check out dependencies and run build `./build.sh`
- Run local development server `npm run dev`
- Access the web site `http://localhost:8080/`
